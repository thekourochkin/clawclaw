## 🎨 ЦВЕТОВАЯ ПАЛИТРА

```css
/* Primary */
--black: #000000    /* Текст, границы, кнопки */
--white: #FFFFFF    /* Фон, текст на черном */

/* Accent */
--accent: #0000FF   /* Pure blue - только для акцентов */

/* Secondary */
--gray: #808080     /* Вторичный текст */
```

**Использование**:
- **90%** - черный и белый
- **8%** - серый (вторичный текст)
- **2%** - синий (только акценты)
